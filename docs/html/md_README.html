0.- Definition

A Fibonacci word is a specific sequence of binary digits (or symbols from any two-letter alphabet). The Fibonacci word is formed by repeated concatenation in the same way that the Fibonacci numbers are formed by repeated addition. But unlike the fibonacci number, Fibonacci word has its first two terms different from each other. Example:

In Fibonacci word,
F1 = "a",
F2 = "b",
F3 = "ab",
F4 = "bab",
F5 = "abbab",
F6 = "bababbab",
F7 = "abbabbababbab".
.....

where F(n) = F(n-2) + F(n-1) and + represents the concatenation of strings.
